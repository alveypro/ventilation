## 🏥 医学内容详解

### 1. 医生中心模块 (5个)

#### 📋 诊断标准
- AASM 2019 睡眠呼吸暂停诊断标准
- AHI分级表 (5个等级，从正常到极重度)
- COPD严重程度分级 (GOLD 2024)
- 无创通气临床指征详表

**关键数据:**
- AHI分级: 正常 < 5 → 轻5-15 → 中15-30 → 重30-60 → 极重>60
- FEV1分级: 正常 ≥ 80% → 轻70-79% → 中50-69% → 重30-49% → 极重<30%

#### 💊 治疗指南
- CPAP vs BiPAP vs VPAP 选择算法 (决策树)
- 患者对CPAP不耐受的5步解决方案
- CPAP压力滴定指南 (4周循序渐进方案)
- 无创通气指征 (6大类型)

**关键指南:**
- 压力滴定: 第1周 4-5 cmH₂O → 第2-3周 6-10 → 第4周+ 8-15
- BiPAP参数: 吸气压 = CPAP压力，呼气压 = 吸气压 - 2-4

#### 📊 临床证据
- CPAP治疗有效性数据:
  - AHI改善: 88-92%
  - 血压改善: 收缩压下降 7-12 mmHg
  - 白天嗜睡改善: ESS评分下降 7-10 分
  - 心血管保护: 心梗风险降低 32-48%
- 关键研究 (3项):
  - APPLES Study: 1156患者，6个月
  - SAVE Trial: 2717患者，3.7年
  - 中国研究: 521患者，12周

#### 👨‍⚕️ 病例讨论
- **病例1**: 中年男性重度OSA
  - 诊断: AHI 78, 高血压, ESS 18
  - 治疗: AutoSet CPAP, 压力12-14
  - 结果 (8周): AHI↓到4-6, 血压↓135/85, ESS↓8-10

- **病例2**: COPD患者呼吸衰竭
  - 诊断: FEV1 28%, Ⅱ型呼吸衰竭
  - 治疗: VPAP ST/iVAPS, 目标潮气量400-500mL
  - 结果 (2周): PaCO₂↓48-50, pH↑7.38-7.40

#### ❓ 医生FAQ (13+)
- CPAP见效时间和患者期望管理
- 患者不耐受的处理对比表
- 云端数据远程患者管理最佳实践

### 2. 患者中心模块 (5个)

#### 🧠 疾病认知
- 什么是OSA (通俗讲解)
- 发生原理 (4步图解)
- 风险因素 (6个卡片):
  - 超重肥胖 (BMI > 25)
  - 中年男性 (患病率是女性2-3倍)
  - 家族史 (风险增加3倍)
  - 吸烟和酒精
  - 颈围增大
- 症状识别 (分夜间、白天、长期)
- 严重程度对比卡片

**关键数据:**
- 全球患者: 9亿人
- 治疗人数: 2亿人

#### ⚙️ 机器使用指南
- 开箱检查清单 (4步)
- 面罩选择和调试:
  - 鼻罩/鼻枕: 舒适、小巧、便携
  - 口鼻罩: 覆盖范围大、可通气
  - 全脸罩: 稳定、适合高压力
  - 调试要点: 紧度、泄漏、压迫
- 基础设置说明:
  - 压力: 医生处方 4-20 cmH₂O
  - 湿度: 4-6 级
  - 温度: 32-35℃
  - Ramp: 启用并逐步升高
- 首次使用 (6步骤):
  1. 插电打开主机
  2. 检查指示灯
  3. 戴上面罩
  4. 启动机器
  5. 调节气流
  6. 试运行15-20分钟
- 睡眠适应 (4周):
  - 第1周: 每晚2-3小时
  - 第2-3周: 每晚4-6小时
  - 第4周+: 每晚7-8小时

#### 🏃 生活管理
- 改善睡眠环境:
  - 保持黑暗 (遮光窗帘)
  - 保持凉爽 (16-19℃)
  - 减少噪音
  - 保持整洁舒适
- 睡眠习惯 (4个建议):
  - 固定上床和起床时间
  - 避免临睡前看手机
  - 避免咖啡因和酒精
  - 坚持每天30分钟运动
- 饮食调整 (4个建议):
  - 晚餐避免油腻
  - 增加纤维素
  - 避免过饱睡眠
  - 睡前3小时不进食
- 体重管理:
  - 减重10% 可改善 AHI 30%
  - 多做有氧运动
  - 循序渐进
- 应该避免 (4个禁忌):
  - 吸烟和酒精 (加重阻塞)
  - 安眠药 (加重暂停)
  - 仰卧睡眠 (侧卧更好)
  - 过度疲劳 (加重打鼾)
- 长期坚持 (5个关键):
  - 坚持使用 (每晚7-8小时)
  - 定期维护 (每周清洁)
  - 定期复诊 (每3-6个月)
  - 症状监测
  - 生活方式改善

#### 🔧 故障排查 (5个常见问题)
1. **面罩泄漏** (噗噗声)
   - 检查戴上是否正确
   - 调整面罩松紧
   - 更换垫垫或面罩
   - 检查管道连接

2. **幽闭恐怖症** (恐惧和紧张)
   - 白天不戴机器，只戴面罩
   - 逐步增加戴面罩时间
   - 使用Ramp功能
   - 进行放松训练
   - 尝试不同面罩
   - 必要时咨询心理医生

3. **无明显改善** (几周仍无效)
   - 使用时间不足 (需>6小时/晚)
   - 压力设置错误
   - 适应期 (需4-12周)
   - 其他医学原因
   - 不良习惯 (吸烟/酒精)

4. **皮肤伤害** (红肿/划痕)
   - 降低面罩紧度
   - 使用护垫
   - 交替面罩使用
   - 皮肤护理 (保湿)
   - 清洁卫生
   - 严重时就医

5. **机器报警**
   - 高泄漏率 → 调整面罩
   - 管道断开 → 检查连接
   - 电源故障 → 检查电源
   - 传感器错误 → 联系售后
