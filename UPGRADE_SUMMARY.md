#### 2. OSA诊断标准库

**AHI分类 (5个等级):**
```
正常:        < 5        无呼吸暂停
轻度:     5-15        偶有打鼾，偶尔嗜睡
中度:    15-30        明显打鼾，白天嗜睡
重度:    30-60        严重暂停，极度嗜睡
极重度:   > 60        危险因素多，并发症风险高
```

**诊断标准 (AASM 2019):**
- 标准A: 症状 (6项中至少1项)
- 标准B: 客观诊断 (PSG或HSAT)
- 标准C: 组合诊断 (A+B或AHI≥15)

**严重程度分级:**
- 轻度: 观察和生活方式改善
- 中度: 强烈推荐CPAP
- 重度: 必须CPAP治疗

#### 3. COPD分类标准库

**FEV1分级 (5个等级):**
```
正常:       ≥ 80%     无气流阻塞
轻度:    70-79%     轻度阻塞
中度:    50-69%     中度阻塞，考虑无创通气
重度:    30-49%     严重阻塞，推荐通气
极重度:   < 30%     极度阻塞，强制通气/有创通气
```

**治疗阶梯:**
- 第1阶段: 生活方式改善
- 第2阶段: 支气管扩张剂
- 第3阶段: 多种药物+无创通气
- 第4阶段: 氧疗+长期通气

#### 4. 无创通气指征库

**6大指征详解:**
```
1. 急性呼吸衰竭
   条件: 急性加重、感染、ARDS
   机型: BiPAP / VPAP
   有效率: > 80% (救命治疗)

2. COPD伴CO2潴留
   条件: pH < 7.35, PaCO2 > 50 mmHg
   机型: BiPAP / VPAP ST / iVAPS
   有效率: 改善生存率 30-50%

3. 神经肌肉病
   条件: ALS、SMA、脊髓灰质炎
   机型: VPAP ST / iVAPS
   有效率: 改善生活质量，延长寿命

4. 胸壁异常
   条件: 脊柱侧弯、后凸
   机型: BiPAP / VPAP
   有效率: 避免呼吸衰竭

5. 肥胖低通气
   条件: BMI > 30, 日间PaCO2 > 45
   机型: BiPAP VAuto / iVAPS
   有效率: 改善通气

6. 心力衰竭
   条件: 心源性肺水肿、Cheyne-Stokes呼吸
   机型: CPAP / ASV
   有效率: 改善心功能，减少再住院
```

#### 5. 选择决策树

**4步骤决策流程:**
1. 第1步: 诊断确认 (6种疾病分类)
2. 第2步: 严重程度评估 (轻/中/重)
3. 第3步: 功能需求 (7个功能选项)
4. 第4步: 预算考虑 (5个价格段)
